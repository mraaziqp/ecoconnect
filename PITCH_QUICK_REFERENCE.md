## 🎯 THE 5-MINUTE PITCH

### Problem Statement
"Family wealth management is fragmented across 5+ tools.
Clients see quarterly reports, not real-time insights.
Your team wastes 30% of time coordinating scattered information."

### Solution
"One unified ecosystem OS where:
- All financial data is in one place
- Clients see real-time portfolio updates
- Role-based access ensures privacy
- AI provides insights, not manual reports
- Families + advisors + accountants collaborate securely"

### Key Differentiators
✓ Purpose-built for family wealth (not generic CRM adapted)
✓ Estate Vault (wills, trusts, inheritance planning)
✓ Advisor Dashboard (professional workspace)
✓ Real-time data (not quarterly reports)
✓ Secure collaboration (encrypted, POPI-compliant)

### Value Proposition
✓ 1 tool replaces 5
✓ 30% team time savings
✓ 50% faster client onboarding
✓ New recurring revenue stream
✓ Modern client experience

### Close
"Let's do a 3-month pilot with your top 5 clients.
We handle setup & migration. You see the ROI in Q1.
When can we start?"

## 🔴 RED FLAGS TO AVOID

❌ Don't over-explain technical details
❌ Don't read slides word-for-word
❌ Don't spend >5 min on any single app
❌ Don't talk about pricing first (talk value first)
❌ Don't make promises about customization you can't keep
❌ Don't let them interrupt the demo flow
✅ DO keep energy high and let their questions guide
